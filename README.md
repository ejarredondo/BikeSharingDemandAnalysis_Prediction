# Information Structures - Bike Sharing Demand Analysis & Prediction

## Project Overview
This repo houses all code and CSVs I used for my SP26 Information Structures final class project. The data was available on Kaggleand I utilized Python libraries of Pandas, Matplotlib and Seaborn to visualize relationships between variables to determine any actionalble insights.

Based on these relationships and patterns I saw, I performed t-tests on notable features to determine their statistical signficance utilizing Python libraries NumPy and SciPy.Stats. 

Following this, I used Scikit-Learn to train multiple prediction models, which i then compared the results of to determine which model was the most accurate.

Technologies Used
- Python
- Jupyter Notebooks
- Pandas
- SciPy
- Scikit-Learn
- Seaborn
- Matplotlib
- NumPy

## How to run this project - 
1. clone repository -
   ```bash
   git clone https://github.com/ejarredondo/BikeSharingDemandAnalysis_Prediction.git
   cd BikeSharingDemandAnalysis_Prediction
   ```
2. create virtual environment (recommended) -
   ```bash
   python -m venv venv
   source venv/bin/activate    # Mac/Linux
   venv\Scripts\activate       # Windows
   ```
3. install dependencies -
   ```bash
   pip install -r requirements.txt
   ```
4. run the notebook -
   open Juypter Notebook:
   ```bash
   juypter notebook
   ```
   Then, open notebook inside `notebooks/` folder
